📖 关于Halo系统

选股逻辑

股票选择（50%）+ 买入时机（50%）

股票选择：IAS评分模型
  • 第一层：行业Alpha (25%) — 景气指数、盈利趋势、资金流、估值
  • 第二层：公司Alpha (30%) — 估值、盈利能力、护城河、全球化能力
  • 第三层：资金Alpha (25%) — 连续资金流、筹码集中度、机构行为
  • 第四层：趋势Alpha (20%) — 截面动量、均线结构、成交量、波动率
  • 事件驱动修正 (±10分)

双重过滤：行业评分 < 50 或 公司评分 < 50 → 直接淘汰

买入时机：五维估值锚定连续赋分模型（总分120分）
  • 历史PE分位 (40分)：`(1-pe_pct)×40` — 越低越便宜
  • 行业PE偏离 (30分)：折价30%满分，溢价50%零分
  • 行业景气度 (30分)：GDP增速(5%)为基准锚
  • PB分位 (10分)：低PB加分
  • 股息率 (10分)：高分红加分

核心原则：越便宜→分越高，线性连续赋分，避免扎堆

7大选股特征
  1. 资产随时间增值，越久越值钱
  2. 产品无论朝代更替都恒久需求
  3. 内生性增长，不需再投入
  4. 高分红率
  5. 中国国有资本投资加仓
  6. 未来社会发展趋势中受益
  7. 符合中国政府政策方向
